# Replace the whole file on GitHub

## Step 1: Open the file on GitHub

1. **GitHub** → your repo → **src** → **lib**
2. Click **dailyGameQuestions.ts**
3. Click the **pencil icon** (Edit this file)

---

## Step 2: Select all and delete

1. In the editor, press **Ctrl+A** (or **Cmd+A** on Mac) to select all
2. Press **Delete** (or **Backspace**) so the file is empty

---

## Step 3: Copy the content below

Open the file **dailyGameQuestions_FULL.txt** in your project (in football-trivia folder). It has the full correct content. Copy all of it (Cmd+A, Cmd+C).

---

## Step 4: Paste on GitHub

1. Click in the empty editor on GitHub
2. Paste (Cmd+V or Ctrl+V)
3. Scroll down and click **Commit changes**

---

Then: **vercel.json** → Edit → replace with the yarn config → Commit. Then Vercel → Redeploy.
